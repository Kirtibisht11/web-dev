lists in html

there are 3 types of lists:

  1. ordered lists
  2. unordered lists
  3. descriptive lists

Ordered lists store data in order or sequential manner
example: list of students enrolled in BCA

  1. kirti
  2. mohit
  3. annant
  4. ram
  5. shreya

unordered lists store data in random order
example: list of subjects in course

  • dbms
  • operating system
  • web application development
  • python programming

descriptive lists store data term and related defition to it
example: info about subjects

dbms
database management system taught by xyz.
operating systems
has multipurposes & taught by abc.
web application development
has front end & back end & taught by qpr.
